# 🌊 LG-KDE

> A KDE Plasma setup inspired by Apple's Liquid Glass design language.

## About

Hi! I'm **Mohamed**, a **13-year-old Linux enthusiast from Egypt 🇪🇬**.

LG-KDE is my personal KDE Plasma configuration focused on creating a clean, modern desktop with translucent panels, smooth blur effects, and a polished glass-like appearance.

This repository contains my KDE configuration files and an installation script to help you recreate the setup on your own system.

## Features

* ✨ Liquid Glass-inspired design
* 🖥️ Custom Plasma panels
* 🧩 Plasma widgets
* 🎨 KDE Plasma configuration
* ⚡ Easy installation with `install.sh`
* 🛠️ Designed for KDE Plasma 6

## Installation

Clone the repository:

```bash
git clone https://github.com/BLEX-TECH/LG-KDE
cd LG-KDE
```

Make the installer executable:

```bash
chmod +x install.sh
```

Run the installer:

```bash
./install.sh
```
